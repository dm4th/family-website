# Mode classifier

## Family mode
Use when the user's emotional goal is connection, memory, identity, belonging, or storytelling.
Common pages:
- family home
- profiles
- photo albums
- event memories
- timeline / history
- travel journals

Questions:
- Is imagery central?
- Is this about people or moments?
- Should the page feel warm and immersive?

## Operations mode
Use when the user is managing real-world assets or tasks.
Common pages:
- properties
- maintenance
- vendors
- reservations
- vehicles
- household logistics
- calendars tied to upkeep

Questions:
- Does the page need scanning and actionability?
- Is spatial or status information primary?
- Should it feel orderly and practical?

## Advisory mode
Use when the user is reviewing financial, legal, trust, or governance information.
Common pages:
- trust overview
- document center
- advisor contacts
- summaries and memos
- distributions and schedules
- meeting notes

Questions:
- Should the page feel formal and calm?
- Is reading and interpretation more important than browsing?
- Should it resemble a briefing or packet?

## Hybrid guidance
Only the top-level dashboard should feel intentionally hybrid.
Most interior pages should have one dominant mode.
