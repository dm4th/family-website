# Page brief template

Before implementing a new page, Claude should produce a short internal brief using this structure:

- Primary mode:
- User intent on this page:
- Dominant module:
- Supporting module 1:
- Supporting module 2:
- Visual tone:
- What to avoid:

## Example
- Primary mode: Operations
- User intent on this page: Review the current state of a property and act on pressing upkeep issues
- Dominant module: Property overview with hero image, facts, and open tasks
- Supporting module 1: Vendor/contact list
- Supporting module 2: Recent expenses and documents
- Visual tone: orderly, spatial, premium, calm
- What to avoid: gallery-like image treatment, bright finance dashboard widgets, equal-sized utility cards
